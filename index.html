Linux Permissions Quiz!

Match the symbolic permission (like rwxr-x---) to its 3-digit octal number (like 750). Remember: r=4, w=2, x=1. Add them up for each section (user, group, other).
Example: -rw-r--r-- is User(rw- = 4+2=6), Group(r-- = 4), Other(r-- = 4) -> 644.
